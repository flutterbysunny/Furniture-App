# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/top_100_flutter_ui.git
```

## 2️⃣ Go to Project Folder

```bash
cd top_100_flutter_ui
```

## 3️⃣ Install Packages

```bash
flutter pub get
```

## 4️⃣ Run App

```bash
flutter run
```
